CARPENTRY TOOLS

Carpentry is the art and craft of shaping wood to create useful objects. Carpenters use a variety of tools to cut, shape, assemble, and finish wood projects.The principle hand tools used in a carpentry workshop can be classified into:

1) Marking and measuring tools

2) Cutting tools

3) Plannning tools

4) Boring tools

5) Striking tools
